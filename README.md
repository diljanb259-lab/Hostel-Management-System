# Hostel Management System (C++)

A simple Hostel Management System developed in C++ using Object-Oriented Programming (OOP) concepts.

## Features

* Add students to hostel
* Allocate hostel beds
* Remove students from hostel
* Display all hostel students
* Store student records using Vector STL

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Vector
* Console Application

## Concepts Used

* Classes and Objects
* Constructors
* Vectors
* Functions
* Encapsulation
* Basic Data Management

## How to Run

### Compile

```bash
g++ main.cpp -o hostel
```

### Execute

```bash
./hostel
```

For Windows:

```bash
g++ main.cpp -o hostel.exe
hostel.exe
```

## Menu

1. Add Student
2. Remove Student
3. Display Students
4. Exit

## Sample Output

